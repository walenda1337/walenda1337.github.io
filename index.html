🧠 Czym jest system operacyjny?

System operacyjny (OS – Operating System) to podstawowe oprogramowanie zarządzające komputerem. Jest pośrednikiem między użytkownikiem a sprzętem. Bez niego komputer nie mógłby uruchamiać programów.

Główne zadania:

  • Zarządzanie procesorem (CPU)
  • Zarządzanie pamięcią RAM
  • Obsługa dysków i systemów plików
  • Kontrola urządzeń (drukarki, karty graficzne)
  • Zarządzanie użytkownikami
  • Zapewnienie bezpieczeństwa

⚙️ Budowa systemu operacyjnego

1. Jądro (Kernel)

Najważniejsza część systemu odpowiadająca za komunikację ze sprzętem.

2. Powłoka (Shell)

Interfejs użytkownika – graficzny (GUI) lub tekstowy (CLI).

3. System plików

Organizacja danych na dysku (NTFS, ext4, APFS).

🪟 Windows

System firmy Microsoft – najpopularniejszy na świecie.

Ważne wersje:

  • Windows 95
  • Windows XP
  • Windows 7
  • Windows 10
  • Windows 11

Zalety:

  • Najlepszy do gier
  • Duża kompatybilność programów
  • Wsparcie producentów sprzętu

Wady:

  • Częste ataki wirusów
  • Płatna licencja

🐧 Linux

System open-source stworzony w 1991 roku przez Linusa Torvaldsa.

Popularne dystrybucje:

  • Ubuntu
  • Debian
  • Fedora
  • Arch Linux
  • Linux Mint

Cechy:

  • Darmowy
  • Bardzo bezpieczny
  • Dominacja na serwerach i superkomputerach

🍎 macOS

System firmy Apple oparty na Unix.

  • Stabilny i zoptymalizowany
  • Popularny w grafice i montażu
  • Działa tylko na komputerach Apple

📱 Systemy mobilne

  • Android – oparty na Linuxie
  • iOS – system Apple dla iPhone
  • HarmonyOS – system Huawei

🔐 Bezpieczeństwo

  • Zapora sieciowa (Firewall)
  • Antywirus
  • Szyfrowanie danych
  • Aktualizacje systemowe
